# example-skill

这是一个示例技能，你可以复制这个目录来创建自己的新技能。

## 适用场景

当用户需要：
- 创建一个新的技能模板
- 了解技能目录怎么组织
- 快速开始编写 `SKILL.md`

## 推荐目录结构

```text
example-skill/
├── SKILL.md
├── templates/
├── scripts/
└── examples/
```

## 你应该在这个技能里写什么

建议至少包含以下内容：
1. 技能用途
2. 触发关键词或触发场景
3. 操作步骤
4. 输入输出格式
5. 注意事项
6. 示例

## 最小示例流程

1. 读取用户需求
2. 判断是否属于本技能处理范围
3. 如果是，则按技能规范执行
4. 输出结果并给出后续建议

## 示例

用户说：
> 帮我创建一个新的自定义技能

你可以：
- 复制当前目录
- 修改 `name`
- 补全 `description`
- 编写具体规则

## 注意事项

- `SKILL.md` 建议写清楚边界，避免技能职责过宽
- 如果依赖脚本/模板，尽量放在技能目录内并相对引用
- 目录结构应保持稳定，方便后续同步到本地 skills 目录
